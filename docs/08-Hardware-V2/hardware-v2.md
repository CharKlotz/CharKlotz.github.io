## Hardware improvements
The overall design of my subsystem is pretty simple, and the BPW34 Photodiode performed well for the use case, so a version 2 would likely be a slight improvement of the current ystem still using that photodiode part. 

## Design Changes
A version 2 might involve using multiple photodiodes in different locations and averaging their signals to get a more accurate reading. This would still involve the same hardware, as the current op-amp being used has 3 more op-amps available. 

## Schematic and PCB Improvements
The schematic and the PCB could both be improved for clarity, for example parts could be more spread out from one another, making it easier to solder and debug the system.
